# Customer Shopping Behaviour Analysis

## Project Overview
This project focuses on analyzing customer shopping behavior to understand purchasing patterns, customer engagement, spending trends, discounts, subscriptions, and product performance.

The goal of this analysis is to derive business insights using Python, SQL, and Power BI to support data-driven decision-making.

---

## Business Problem Statement

A retail company wants to better understand customer shopping behavior to improve sales performance, customer satisfaction, and long-term loyalty.

The company wants answers to questions like:

- Which customer segments contribute more revenue?
- How do discounts influence spending?
- Are subscribed customers spending more?
- Which products are highly purchased?
- Which age groups contribute the most revenue?

The objective is to identify trends and improve marketing and product strategies.

---

## Project Objectives

- Clean and preprocess customer shopping data
- Perform exploratory data analysis using Python
- Solve business problems using SQL queries
- Build an interactive Power BI dashboard
- Generate actionable business insights

## Project Workflow

### 1. Data Cleaning & Preparation (Python)

### 2. SQL Business Analysis
Solved business questions such as:

1. Revenue generated by male vs female customers  
2. Customer spending after discounts  
3. Top purchased products  
4. Standard vs Express shipping comparison  
5. Subscription spending behavior  
6. Customer segmentation analysis  
7. Repeat customer behavior  
8. Revenue contribution by age group  

### 3. Dashboard Creation (Power BI)

- Revenue trends

- Customer segmentation
- Age group analysis
